Reading question types<\/h2>\n\n\n\n

Here are the types of Reading questions that you are likely to encounter in your test. Understanding and mastering these question types is very important if you want to do well in the Reading test!<\/p>\n\n\n\n

    \n
  1. Matching headings: This type of task requires you to choose a heading from a list of headings to match a section or paragraph in the reading passage.\u00a0<\/span><\/li>\n\n\n\n
  2. Complete the summary: This type of question asks you to complete a summary guess by filling in the blanks with words that can be found in the passage or given words. <\/li>\n\n\n\n
  3. Choose a list of answers: This question type requires you to choose a list of answers from a given list of choices. Note that this format is different from multiple choice, because all the answers in this format are only for one question.<\/li>\n\n\n\n
  4. Sentence matching: In this type of question you will be given a list of incomplete sentences without ending and a list of the remaining parts of the sentence. Your job is to match incomplete sentences with appropriate endings based on the passage.<\/li>\n\n\n\n
  5. Complete the chart: This type of question requires you to complete the chart in the form of labeling. The example diagram could be a model\/sketch of an object described in the article.<\/li>\n\n\n\n
  6. True\/False\/Not Given: This type of task asks you to decide whether the information given in the question is True or False if it is mentioned in the passage, or Not Given if it does not exist, or reading does not provide enough information to infer this.<\/li>\n\n\n\n
  7. Complete sentence: This type of task requires you to complete the sentence as a fill in the blank with words that can be found in the passage.<\/li>\n\n\n\n
  8. Choose the most appropriate title: This type requires you to choose the most suitable title from the given titles.<\/li>\n\n\n\n
  9. Complete table: This type of task requires you to complete a table based on information taken from the text.<\/li>\n\n\n\n
  10. Short answer: This type of test requires you to briefly answer the given questions (maybe no more than 1 word\/2 words\/3 words…) based on the information contained in the passage.<\/li>\n\n\n\n
  11. Matching paragraph information: This type of task requires you to match the information given in the question with the corresponding information that can be found in the passage. <\/li>\n\n\n\n
  12. Multiple-choice: This type of question requires you to choose the correct answer to a given question, usually from 3 or 4 options. <\/li>\n\n\n\n
  13. Classify: This format gives you a list of categories and a list of information. You must decide which category the given information corresponds to.<\/li>\n\n\n\n
  14. Complete flowchart: This type of task requires you to complete the flowchart with the words given in the passage. Please note that this type of lesson is different from the completed chart form, because the flowchart will represent a certain process (process), ie successive steps. <\/li>\n<\/ol>\n","protected":false},"excerpt":{"rendered":"
